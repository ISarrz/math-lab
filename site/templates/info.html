Список функций

Геометрия:
  1. Площадь треугольника
    функция принимает 3 аргумента - стороны треугольника. Если треугольник с заданными сторонами существует, то
    функция вернёт значение его площади в виде числа.
  2. Площадь выпуклого четырехугольника
    3. функция принимает 4 аргумента - стороны выпуклого четырехугольника.
     функция вернёт значение его площади в виде числа.
  3. Площадь круга
    4. функция принимает 1 аргумент - радиус круга.
     функция вернёт значение его площади в виде числа.
  4. Вычисление внутреннего угла выпуклого n угольника
    5. функция принимает 1 аргумент - число сторон выпуклоно n уголника.
     функция вернёт значение внутреннего угла многоугольника в градусах.
Алгебра:
  1. Разложение на простые множители
    2. функция принимает 1 аргумент - число.
     функция вернёт список всех простых множителей и их степеней(если они существуют).
  2. Вычисление иррациональных корней
    3. функция принимает 1 аргумент - число.
     функция вернёт иррациональное число.
  3. Сокращение дробей
    4. функция принимает 2 аргумент - числитель и знаменатель.
     функция вернёт иррациональное число.
  4. Решение линейных уравнений с целыми индексами
  5. Решение квадратных уравнений с целыми индексами
  6. Определение свойств линейной функции с целыми индексами
  7. Определение свойств квадратичной функции функции с целыми индексами
  8. Вычисление НОК и НОД
